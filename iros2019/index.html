RoFICoM – What Does It Do?

It connects robots:

  • mechanically
  • communicatively
  • electrically

Why should I Use It?

RoFICoM is

  • versatile
  • adaptible
  • encapsulated
  • available

Fits many use cases.

  • flat & compact
  • genderless & grid-aware connection
  • no power required for sustaining the connection
  • support for passive counterparts

Arbitrary binary packets can be exchanged.

  • support for practically any packet-based procol
  • TCP/IP as the main communication protocol
  • simple protocols possible for simple peripherals

Parametric model

  • custom diameter or
  • retract distance can be generated

Easy to integrate, drop-in upgrades.

  • fixed mounting holes
  • electrical connection via pogo pins
  • stand-alone device accessible via an SPI bus

Existing solutions are not commonly available

You can read a paper, but you cannot buy them or download blueprints.

RoFICoM is Open-Hardware

  • optimized for 3D printing/polyurethane molding
  • (relatively) easy to build
  • no special components
Blueprints, sources and assembly guide available at:
https://rofibots.com/connector/
